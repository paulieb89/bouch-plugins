# Example Queries

Copy-paste prompts for common HMRC tax questions.

## VAT

```
What is the VAT rate for [good/service]?
```

```
My client provides [describe service]. Are they liable to register for VAT?
```

```
Is the supply of [item] zero-rated or exempt? What is the practical difference?
```

```
My client has opted to tax their commercial property. What does this mean for their tenants?
```

## Making Tax Digital

```
Check MTD status for VRN [9-digit number, no GB prefix].
```

```
What are the current MTD for VAT obligations for a business with turnover of £X?
```

```
When does MTD for Income Tax apply and to whom?
```

## Income Tax & Property

```
What allowable expenses can a residential landlord claim?
```

```
How does section 24 affect a higher-rate taxpayer landlord with mortgage interest of £X?
```

```
My client is selling a buy-to-let property. What CGT do they owe and when must they report it?
```

```
What is the tax treatment of furnished holiday lettings from April 2025?
```

## Corporate Tax

```
What is the current corporation tax rate for a company with profits of £X?
```

```
What are the R&D tax credit rules for SMEs?
```

```
My client is considering a share buyback. What are the tax implications?
```

## General Guidance

```
Search HMRC guidance on [topic] and summarise the key rules with current thresholds and rates.
```

```
What is HMRC's position on [arrangement]? Is there published guidance?
```
