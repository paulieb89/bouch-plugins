# Starter Prompts

Copy-paste prompts for common legal research tasks using the UK Legal MCP server.

## Legislation

```
What does section [N] of the [Act Name] [Year] say? Is it in force?
```

```
Get me the table of contents for the [Act Name] [Year].
```

```
What did section [N] of the [Act Name] say on [date]?
```

```
Find all provisions in the [Act Name] that relate to [topic].
```

## Case Law

```
Find [Supreme Court / Court of Appeal / High Court] cases on [legal issue] after [year].
```

```
Search for cases on [topic] in the [court code] court.
```

```
What did the court say about [specific point] in [Case Name]? Search the judgment for [keyword].
```

```
Has [Case Name] [neutral citation] been approved or distinguished? Show me its citation network.
```

## Statutory Interpretation (Pepper v Hart)

```
Search Hansard for debates on [Act Name] specifically about [provision or term].
Did any minister explain what "[phrase]" meant?
```

```
What was the parliamentary reception of the [Bill Name]? Were there significant objections?
```

## Full Research Brief

```
Research the law on [topic] and produce a brief covering:
- The relevant statutory provisions with section text
- Key cases with neutral citations
- Any relevant Hansard debates on legislative intent
- A plain English summary of the current legal position
```

## Firm AI Rollout (internal use)

```
Draft a precedent for [document type] based on the following statutory requirements: [paste sections]
```

```
Check whether this clause is consistent with the current law: [paste clause]
```

```
Summarise the key legal risks in this situation: [describe facts]
Cite the relevant provisions and cases.
```
