# Pitch Brief Template

```
## Pitch Brief: [Company Name]

### Snapshot
[2-3 lines: what they do, approximate size, location, company status, who runs it]

### Recent signals
- [3-5 bullets from the web pass: hires, campaigns, launches, press, public statements, funding]
- [Each bullet should be a specific, dateable event — not "they are growing" but "announced Bristol studio opening, March 2026"]

### Who controls it
[Key director(s) and PSC, anything notable. One or two lines. If PSC is a foreign parent, name it here.]

### Health check
[Filing status, any gazette notices, disqualified director flags, or "Clean — active company, filings current, no red flags."]

### Person notes
[Only if contact name was provided. 3-4 lines, public info only — articles written, podcasts, conference talks, press quotes. 
If no contact provided, write: "No contact name provided. Likely role to target: [function] — search LinkedIn for the parent company name + that function."]
[Skip this section entirely only if no contact was provided AND no guidance is needed on who to approach.]

### Hooks for outreach
- [3-5 specific things that could open a cold email, in rough order of strength]
- [Hook 1 should be the strongest — most recent, most specific, most tied to what the user is pitching]
- [Hook 5 can be a fallback — something structural or biographical if recency is thin]

### Watch out for
- [Red flags, sensitivities, recent problems, things NOT to mention]
- [Almost always populate this section — even structurally clean companies have operational sensitivities: communication style, decision locus, incumbent relationships, timing constraints, areas that would read as amateurish or uninformed.]
- [Only omit if you genuinely cannot identify anything, which is rare.]

### Sources checked
[One line: "Companies House, Gazette, web pass (DD Month YYYY)." Be explicit about what was and was not used. If a tool failed, note it here.]
```
